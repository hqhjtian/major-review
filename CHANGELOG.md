## V1.5 - AI 出题 Prompt 与正确率判定

### 新增
- 新增 AI 出题 Prompt 区域。
- 支持根据当前知识点生成不同类型的出题提示词：
  - 选择题 Prompt
  - 填空题 Prompt
  - 简答题 Prompt
  - 综合自测 Prompt
- 支持一键复制 Prompt，复制后可发送给 DeepSeek / ChatGPT 生成练习题。
- 新增练习结果录入区。
- 支持手动输入练习正确率，并根据正确率自动更新当前知识点掌握状态：
  - 正确率 ≥ 80%：标记为“掌握”
  - 50% ≤ 正确率 < 80%：标记为“模糊”
  - 正确率 < 50%：标记为“不会”

### 设计说明
- 当前版本暂不直接接入 DeepSeek API。
- 先采用“生成 Prompt → 复制给 AI → 手动输入正确率”的方式，避免 API 配置、费用和隐私复杂度。
- 该功能主要服务于专业课期末复习中的选择题、填空题、简答题和综合自测训练。

## V1.6 - 本地复习本书架

### 新增
- 新增“我的复习本书架”。
- 支持为每套复习资料设置复习本名称。
- 每次解析一套新资料时，自动保存为一本新的复习本，避免覆盖旧资料。
- 支持在本地保存多本复习资料。
- 支持点击书架中的复习本，切换回对应资料继续复习。
- 每本复习本独立保存：
  - 原始复习资料
  - 拆分后的知识板块
  - 当前学习位置
  - 掌握 / 模糊 / 不会状态
  - 车轮复习队列
  - 已完成章节提示记录
- 支持手动保存当前复习本。
- 支持新建空白复习本。
- 支持删除不再需要的复习本。
- 书架中显示每本复习本的基本统计：
  - 总板块数
  - 掌握数
  - 模糊数
  - 不会数
  - 最近更新时间

### 设计说明
- 该版本解决了此前“新资料覆盖旧资料”的问题。
- 现在工具从“单本复习页面”升级为“本地复习书架”。
- 数据仍保存在浏览器本地 localStorage 中。
- 换浏览器、清缓存或换设备仍可能导致数据丢失，因此后续仍需要开发导出 / 导入备份功能。
